1️⃣ Basic Commands
1. man

Shows manual/help for a command.

mca@MCA04:~/Desktop/linux$ man ls

Sample Output (first lines):

LS(1) User Commands
NAME
       ls - list directory contents
2. pwd

Shows current directory.

mca@MCA04:~/Desktop/linux$ pwd

Output:

/home/mca/Desktop/linux
3. ls

Lists files and folders.

mca@MCA04:~/Desktop/linux$ ls

Output:

file1.txt  file2.txt  folder1
4. echo

Prints text.

mca@MCA04:~/Desktop/linux$ echo "Hello Linux"

Output:

Hello Linux
5. read

Reads user input.

mca@MCA04:~/Desktop/linux$ read name
Rahul
mca@MCA04:~/Desktop/linux$ echo $name

Output:

Rahul
